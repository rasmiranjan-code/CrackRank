# CrackRank
CrackRank 🚀  CrackRank is an all-in-one competitive exam preparation platform built with Flutter, Firebase, and Cloud Firestore, designed for exams such as NEET, JEE, CUET, Board Exams, BSc Entrance, and future exam categories.  The platform combines learning, practice, testing, performance tracking, NCERT resources, and student collaboration.
